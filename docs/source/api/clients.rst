Client Classes
=============

The Cohere Compass SDK provides client classes for interacting with different parts of the Compass API:

* ``CompassClient``: Main client for managing indexes and documents
* ``CompassParserClient``: Client for parsing and processing documents
* ``RbacClient``: Client for role-based access control
