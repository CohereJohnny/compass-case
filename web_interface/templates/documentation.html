Documentation

Getting Started

This web interface provides a user-friendly way to interact with the Cohere Compass SDK. The SDK is designed to help you build and manage retrieval-augmented applications.

Key Features

  • Create and manage indexes
  • Upload documents to indexes
  • Search indexed documents
  • Chat with indexes using Cohere's powerful language models
  • API explorer for testing SDK functionality

Prerequisites

To use this interface, you'll need:

  1. A Cohere API key (for chat functionality and LLM access)
  2. A Compass API Bearer token (for accessing the Compass service)

Setting Up

Before using the interface, you should configure your API keys in the Settings page.

Guide to Key Features

1. Index Management

Navigate to the Indexes page to view all your Compass indexes. From there, you can:

  • Create new indexes
  • View index details
  • Upload documents to existing indexes
  • Search indexed documents
  • Chat with specific indexes

2. Document Upload

When uploading documents to an index, you can:

  • Upload text files directly
  • Upload documents in batches
  • Add document metadata

3. Search

The search functionality allows you to find documents in your indexes using:

  • Keyword search
  • Semantic search
  • Filtered search using metadata

4. Chat

The chat interface provides a way to have conversations with your indexed data, powered by Cohere's language models. Key features include:

  • Contextual conversations based on your data
  • Visibility into the retrieved context for transparency
  • Model selection for different capabilities

API Explorer

The API Explorer provides a GUI for interacting with the Compass SDK methods. This is useful for:

  • Testing functionality before implementing it in your code
  • Understanding the API structure and response format
  • Debugging issues with your indexes or documents
